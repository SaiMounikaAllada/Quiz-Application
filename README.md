# Quiz-Application
Java Quiz Application
A console‑based Quiz Application built using Core Java and Object‑Oriented Programming (OOP) concepts.
The application allows users to answer quiz questions using option numbers or direct text input.

📌 Features

Multiple‑choice quiz with predefined questions
Users can answer in three ways:

By entering option number (1–4)
By entering answer text directly
By mixing option numbers and text answers


Case‑insensitive answer checking
Score calculation at the end of the quiz
Clean separation of responsibilities using OOP


🛠️ Technologies Used

Java (Core Java)
OOP Concepts
Scanner class for user input
